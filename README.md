# MarryUs
### 비트캠프 파이널 프로젝트
    구성원 : 정인용, 오수진, 김은비, 임유신, 이승재
    제작의도 : 예비 신랑&신부가 플래너를 통하여 업체를 선정하는것이 아닌 
              신랑&신부가 원하는 플래닝을 역경매를 통하여 업체가 입찰을 할수있는 웹사이트
